# Day 1 - Python Basics

## 📌 Objective

Learn the fundamentals of Python programming by solving basic logic-building problems using variables, user input, conditional statements, and loops.

---

## 📚 Topics Covered

- Variables
- Data Types
- User Input
- Arithmetic Operators
- Comparison Operators
- Logical Operators
- Conditional Statements (`if`, `elif`, `else`)
- While Loop

---

## 💻 Programs Practiced

1. Student Information Program
2. Voting Eligibility Checker
3. Member of Parliament Eligibility Checker
4. Student Grade Calculator
5. Divisibility Checker
6. College Admission Eligibility
7. Customer Membership Level
8. Bank Withdrawal System
9. Digit Counter
10. Reverse a Number
11. Sum of Digits
12. Fibonacci Series (Practice)

---

## 🎯 Learning Outcomes

- Learned how to take input from users.
- Practiced writing decision-making programs.
- Improved logical thinking using conditional statements.
- Understood how loops work for repetitive tasks.
- Solved beginner-level real-world Python problems.
